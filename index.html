Tags have angle brackets, < and >, which <wrap> around text to make elements.
The frontslash character leans forward: /
Elements can have attributes:
This span has a style attribute.
p tags are for paragraphs. strong & em tags make text bold & italic, respectively. Similarly, you can use the sup and sub tags to make text superscript or subscript.
Another cool thing you can do is use q tags for short quotes.

You should always use blockquote tags for longer quotations, a --
-- br tag for a line break, and an hr tag for a horizontal rule.
These tags are self-closing, as there is no matching closing tag.

FYI (hover me!), the abbr tag is for abbreviations.
The cite tag is for Titles of Works by some author.
You can also show text that has been deleted or inserted; or you can just use the s tag to make a strikethrough.
